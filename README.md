# Triple-Trigger-Magnetic-Smart-Switch-for-Low-Power-Applications
Assembly and Embedded C code


This work presents a low-power magnetic switching system designed for controlling electrical loads up to 45 W using a contactless activation mechanism. The system is based on the AT89C2051 microcontroller and incorporates a Hall effect sensor for reliable magnetic field detection. A unique multi-trigger logic is implemented, where the load is activated only after three consecutive detections of a magnet near the sensor within a specified time interval, thereby enhancing operational safety and preventing accidental switching.
The elimination of mechanical components improves system durability and reduces maintenance requirements. The proposed design is compact, energy-efficient, and suitable for applications requiring secure and touchless control. Potential use cases include smart home systems, industrial automation, and restricted-access power control environments.
